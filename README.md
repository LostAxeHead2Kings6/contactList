## Usage

  > Users keep a list of all of their contacts, and those contact's names, email addresses and phoneNumbers. Entries can be updated, or deleted. New entries can always be added.

## Development

1. Clone down this project to your local machine
2. npm install
3. for dev environment initiliaze a local mongoDb server with a database and schema that can be found
   in the database folder
4. To Locally Host,
           npm run server-dev
